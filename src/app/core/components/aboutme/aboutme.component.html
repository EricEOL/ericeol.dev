Sobre mim

  • Back-end

    Como back-end eu atuo principalmente com Java e NodeJS, com foco na construção de API's RESTFULL. Com Java, utilizo bastante nos meus projetos o framework Spring e claro todas as suas ramificações como Spring Data, Cache, Security entre outros. Já com NodeJS, o principal framework que utilizo é o Express.

  • Front-end

    Como front-end eu tenho prazer de utilizar em meus projetos ReactJS e Angular, além é claro do Javascript Vanilla que é a raiz de tudo. Tenho diversos projetos com esses frameworks/libs. Na parte de estilização com CSS, também utilizo diversos frameworks como Bootstrap, Styled Components, Chakra UI, Tailwind CSS e Sass.
